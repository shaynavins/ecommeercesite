# Lynq Generated Backend

This backend was generated by Lynq and connects directly to the PostgreSQL database created during deployment.

## Run locally

```bash
npm install
cp .env.example .env
# Edit DATABASE_URL in .env
npm run dev
```

## Routes

- categories: GET/POST /categories, GET/PUT/DELETE /categories/:id
- products: GET/POST /products, GET/PUT/DELETE /products/:id

## Health check

```bash
curl http://localhost:3001/health
```

## Production notes

- Set `API_BEARER_TOKEN` before exposing this backend publicly.
- Set `CORS_ORIGINS` to your deployed frontend origin(s).
- Keep `DATABASE_URL` and `API_BEARER_TOKEN` in your deployment secret manager, not in git.
- Run `npm run build` and `npm test` before deploying.
